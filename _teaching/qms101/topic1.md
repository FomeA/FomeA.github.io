# Introduction

Statistics is the study of how to **work with data.**

Today, almost every organization uses data to make decisions.\
Businesses use data to understand customers, control costs and plan for the future.

In this course, students will learn how to **collect, organize, understand and interpret data.**

# **Why Study Statistics?**

Think about the following situations:

-   A bank must decide whether to give a loan to a customer.

-   A shop owner must decide how much stock to order.

-   A company wants to know if customers are satisfied.

-   Government wants to know unemployment levels.

All these decisions require **information based on data.**

Statistics helps transform:

**Raw data → Useful information → Better decisions**

# Meaning of Statistics

Statistics has two meanings.

## Statistics as Numbers

Sometimes statistics means **numerical facts.**

Examples:

-   Inflation rate is 4%

-   Average salary is TZS 800,000

-   Company profit is TZS 2 million

These numbers describe real situations.

## Statistics as a Subject

Statistics is also a **field of study.**

It involves:

-   Collecting data

-   Organizing data
 
-   Presenting data

-   Analyzing data

-   Making conclusions

# Branches of Statistics

## Descriptive Statistics

Descriptive statistics summarizes and presents data using:

-   Tables
-   Graphs
-   Averages

Example: A company summarizes monthly sales using a bar chart.

## Inferential Statistics

Inferential statistics uses sample data to make conclusions about a population.

Example: A survey of 100 customers is used to understand opinions of all customers.

# Variables

In statistics, we collect information about people, businesses or objects. The type of information we collect is called a **variable.**

A variable is **anything we measure, count or record.**

### Simple Examples

If we study students in this class, we can collect:

-   Age of each student
-   Gender of each student
-   Amount of money spent on lunch
-   Number of siblings

Each of these is a **variable** because the value is not the same for everyone.

For example:

| Student | Age | Lunch Cost |
|---------|-----|------------|
| A       | 20  | 3000       |
| B       | 22  | 2500       |
| C       | 19  | 2000       |

We can see that:

-   Age is different for each student
-   Lunch cost is different for each student

Therefore, **Age and Lunch Cost are variables.**

**Important Idea**

A variable is simply **the question we are asking about each person or object.**

Examples:

-   “How old are you?” → **Age** is the variable
-   “How much did you spend?” → **Cost** is the variable
-   “What is your gender?” → **Gender** is the variable

## Types of Variables

Once we know what a variable is, the next question is: **what kind of variable is it?**

Variables are mainly of two types:

### Qualitative (Categorical) Variables

These variables **describe a category or quality**. They are not numbers you can do arithmetic with.

Examples:

| Variable       | Possible Values               |
|----------------|-------------------------------|
| Gender         | Male, Female                  |
| Religion       | Christianity, Islam, Hinduism |
| Marital Status | Single, Married, Divorced     |

**Key Idea:**

-   Qualitative variables **tell us which group or category something belongs to**, not a number.
-   Ask yourself: *“Can I count it meaningfully or do I just put it in a group?”* If it’s a group → Qualitative

### Quantitative (Numerical) Variables

These variables **represent numbers that we can measure or calculate**.

Examples:

| Variable                | Example Values              |
|-------------------------|-----------------------------|
| Height (cm)             | 160, 172, 185               |
| Monthly Income (TZS)    | 500,000; 800,000; 1,200,000 |
| Number of Products Sold | 5, 10, 12                   |

**Key Idea:** Quantitative variables are numbers you can **add, subtract, average, or compare**.

## Discrete and Continuous Variables

After understanding **qualitative and quantitative variables**, we now focus on **types of quantitative variables**.

Quantitative variables can be **Discrete** or **Continuous**.

### Discrete Variables

Discrete variables are **countable numbers**. You can list each possible value, and there are **gaps between values**. Examples

| Variable                         | Example Values |
|----------------------------------|----------------|
| Number of students in a class    | 20, 21, 22     |
| Number of products sold in a day | 5, 10, 12      |
| Number of cars in a parking lot  | 0, 1, 2, …     |

**Key Idea:** You can **count them exactly**.

**Visual Tip:** Imagine dots representing each item. You cannot have “half a student” or “half a car.”

### Continuous Variables

Continuous variables are **measurable numbers**. They can take **any value within a range**, including fractions or decimals. Examples in Business

| Variable                 | Example Values      |
|--------------------------|---------------------|
| Height of a student (cm) | 160.2, 172.5, 185.0 |
| Weight of products (kg)  | 2.5, 3.1, 4.0       |
| Daily sales amount (TZS) | 3500.50, 4200.75    |

**Key Idea:** Continuous variables can be **measured as accurately as needed**.

**Visual Tip:** Imagine a ruler — you can measure anywhere along the line, not just whole numbers.

### Quick Student Tip

Ask yourself:

-   “**Can I count the exact** number of items?” → Discrete
-   “Can it take **any value in a range**?” → Continuous

### Think About It

Classify the following as **Discrete** or **Continuous**:

1.  Number of customers in a shop
2.  Monthly income of employees
3.  Number of sales per day
4.  Weight of parcels delivered

### Important Note

Some quantitative variables are **tricky**:

-   Number of likes on a social media post → Discrete
-   Temperature → Continuous
-   Rating on a scale of 1–5 → Discrete (even though numeric)

# Data Collection

Once we know what variables are, the next step is to **collect data**.

Data is **information we gather about people, businesses, or objects**.

There are **two main types** of data: **Primary** and **Secondary**.

## Primary Data

Primary data is **collected directly by you**.

### Methods:

| Method | Description | Example |
|----|----|----|
| Questionnaire | Written set of questions | Survey employees about job satisfaction |
| Interview | Asking questions directly | Ask customers why they prefer a product |
| Observation | Watching and recording | Count the number of customers visiting a store |
| Experiment | Controlled testing | Test two pricing strategies to see which sells more |

**Key Idea:** You collect **fresh information specifically for your study.**

## Secondary Data

Secondary data is **already collected by someone else**.

### Sources:

| Source             | Example                               |
|--------------------|---------------------------------------|
| Government reports | National employment statistics        |
| Books              | Market research publications          |
| Internet           | Online databases and company websites |

**Key Idea:** You **reuse existing data**, which saves time and cost.

# Data Presentation

After collecting data, we must **present it so others can understand it quickly**.

Good presentation makes it **easy to see patterns and compare numbers**.

## Tables

Tables are **rows and columns that organize data**.

Example:

| Student | Age | Lunch Cost (TZS) |
|---------|-----|------------------|
| A       | 20  | 3000             |
| B       | 22  | 2500             |
| C       | 19  | 2000             |

**Tip:** Use tables to show **exact numbers** or **compare values**.

## Bar Charts

Bar charts are **visual tools to compare categories**.

-   Each bar represents a category
-   Height of bar = value or frequency

Example: Comparing number of students spending different amounts on lunch.

**Key Idea:** Bars help **see differences at a glance**.

## Frequency Tables

A frequency table shows **how often values occur**.

Example: Number of students in each lunch cost range.

| Lunch Cost (TZS) | Frequency |
|------------------|-----------|
| 2000–2499        | 3         |
| 2500–2999        | 5         |
| 3000–3499        | 7         |
| 3500–3999        | 3         |

**Tip:** Use frequency tables for **large data sets** to summarize information.

## Histograms

Histograms are **similar to bar charts but for continuous data**.

-   Bars touch each other
-   Width = range of values
-   Height = frequency

Example: Showing distribution of daily lunch costs.

**Key Idea:** Helps **see the shape of data** (e.g., most students spend around 3000–3500 TZS).

## Quick Tips

-   Use **tables** for exact numbers
-   Use **bar charts** for categories or discrete variables
-   Use **histograms** for continuous data
-   Always **label axes and include units**

# Frequency Distribution

After collecting and presenting data, the next step is to **summarize it** using a **frequency distribution**.

A frequency distribution shows **how often each value or range of values occurs**. It helps us **see patterns and trends** quickly.

## Why Use Frequency Distribution?

-   Makes large datasets **easier to understand**
-   Helps identify the **most common values**
-   Prepares data for **charts like histograms and bar graphs**

## Step-by-Step Example

Suppose a lecturer records **daily transport costs (in TZS) for 20 students**:

```         
2000, 2500, 3000, 2000, 4000, 3500, 3000, 2500, 2000, 4500, 
5000, 3000, 2800, 2600, 3200, 2700, 2900, 3100, 3300, 3400
```

### Step 1: Find Minimum and Maximum

-   Minimum = 2000 TZS
-   Maximum = 5000 TZS

### Step 2: Decide Number of Classes

-   Let’s use **6 classes**.

### Step 3: Calculate Class Width

-   Class width = (Maximum − Minimum) ÷ Number of classes
-   Class width = (5000 − 2000) ÷ 6 ≈ **500**

### Step 4: Create Class Intervals

| Transport Cost (TZS) | Frequency |
|----------------------|-----------|
| 2000 – 2499          | 3         |
| 2500 – 2999          | 5         |
| 3000 – 3499          | 7         |
| 3500 – 3999          | 1         |
| 4000 – 4499          | 1         |
| 4500 – 5000          | 3         |

### Step 5: Interpret the Data

-   Most students spend **3000 – 3499 TZS** on transport.
-   Very few students spend **above 4000 TZS**.
-   The data is **concentrated in the middle range**.

## Cumulative Frequency

Cumulative frequency shows **the running total of students up to a certain class interval**.

| Transport Cost (TZS) | Frequency | Cumulative Frequency |
|----------------------|-----------|----------------------|
| 2000 – 2499          | 3         | 3                    |
| 2500 – 2999          | 5         | 8                    |
| 3000 – 3499          | 7         | 15                   |
| 3500 – 3999          | 1         | 16                   |
| 4000 – 4499          | 1         | 17                   |
| 4500 – 5000          | 3         | 20                   |

**Tip:** Cumulative frequency is useful for finding **medians, quartiles, and percentiles**.

## Visualizing Frequency Distribution

### Bar Chart

-   Each bar represents a **class interval**
-   Height = **frequency**

### Histogram

-   Bars **touch each other** (continuous data)
-   Width = **class interval**
-   Height = **frequency**

Both charts help students **see patterns easily**.
