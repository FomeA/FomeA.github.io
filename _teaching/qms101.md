<h2> Course Description </h2>
<p>
This course introduces students to the fundamental concepts and techniques of Statistics, with emphasis on practical applications in research and business. It equips learners with essential skills in data collection, analysis, interpretation, and decision-making using statistical methods.
</p>

<h2> Learning Outcomes </h2>
<p>By the end of the course, students will be able to:</p>
  <ul>
<li> Understand and describe different types of data, data collection, and sampling methods </li>
<li> Organize and present data using tables and charts </li>
<li> Compute and interpret summary statistics </li>
<li> Apply probability rules and theoretical distributions </li>
<li> Perform basic statistical analyses including regression and correlation </li>
<li> Estimate population parameters and conduct hypothesis testing </li>
<li> Interpret statistical results for decision-making </li>
    </ul>
