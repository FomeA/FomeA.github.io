MAT 103 – Numerical Analysis I

Approximation, Computation, Modelling & Scientific Problem Solving

Course Description

This course introduces students to numerical techniques for solving mathematical problems that cannot be solved analytically. It covers error analysis, root finding, interpolation, numerical integration, and solving systems of linear equations. Students also use MATLAB and MAPLE for computation.
